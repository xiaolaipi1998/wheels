#### 开始使用
1、安装
使用本架构前，请在css中开启border-box
*{box-sizing: border-box;}
IE8以上浏览器都支持此样式
需要用户自行设置css
        :root {
            --button-height: 32px;
            --font-size: 14px;
            --button-bg: white;
            --button-active-bg: #eee;
            --border-radius: 4px;
            --color: #333;
            --border-color: #999;
            --border-color-hover: #666;
        }
        IE15以上支持此样式
2、安装 wheels
npm i gavin-test-1
3、引入 wheels
  import {Button, ButtonGroup, Icon} from 'gavin-test-1'
  import 'gavin-test-1/dist/index.css'

4、引入svg symbol
